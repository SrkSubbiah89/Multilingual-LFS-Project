# LFS Conversational AI

A multilingual conversational AI system for Labour Force Survey (LFS) data.

## Project Structure

```
lfs-conversational-ai/
├── backend/
│   ├── agents/       # Conversational AI agents
│   ├── api/          # API endpoints
│   ├── auth/         # Authentication & authorization
│   ├── database/     # Database models and connections
│   ├── llm/          # LLM integration and prompts
│   ├── rag/          # Retrieval-Augmented Generation pipeline
│   └── tests/        # Backend tests
├── frontend/
│   ├── components/   # Reusable UI components
│   ├── pages/        # Application pages
│   └── public/       # Static assets
├── docker/           # Docker configuration files
├── data/
│   ├── isco-08/      # ISCO-08 occupation classification data
│   ├── simulated/    # Simulated LFS data
│   └── embeddings/   # Precomputed vector embeddings
├── notebooks/        # Jupyter notebooks for exploration & analysis
└── README.md
```
